# Personal Registration Schedule

## Getting Started
1. Install packages:
  ~~~Shell
  pip install pandas
  pip install mmap
  ~~~
2. Upload a file to InputData named by 'file.txt':
>  the file must be structured line by line:<br>
>  Name<br>
>  City<br>
>  ZIP Code<br>
>  Phone<br>

3. Run script (/path/Personal_Registration_Schedule/):
 ~~~Shell
 ./main.py
 ~~~
