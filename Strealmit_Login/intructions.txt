Executar Local (em STREAMLIT_LOGIN/):
    - Ativar venv: 
        $ venv/Scripts/Activate.ps1

    - Run:
        $ streamlit run src/App.py

    - Desativar venv:
        $ deactivate


Modelagem:
    - .streamlit/config.toml: arquivo de configuração do app.
    - .streamlit/secrets.toml: arquivo de configuaração de credenciais de acesso (não usado).
    - src/Classes/User.py: arquivo de classe criada para manipular os dados com os métodos e acessos necessários.
    - src/Database/database.db: banco de dados em sqlite para inserir, consultar e/ou deletar usuários.
    - src/Packages: módulos criados para gerenciar funções genéricas.
    - src/page/*.py: páginas criadas para separar a tela de login, cadastro e admin.
    - src/App.py: arquivo main.
    - venv: ambiente de desenvolvimento virtual para isolar as dependências do projeto (boa prática de organização de projeto).
